Medcare

An android application which functions as an all in one medical system as it assists the user in diagnosing diseases, finding suitable medical centers , alternate medicines, booking appointments and so on.

Description

...Under Development...

The main aim of this app is to assist the user in diagnosing their diseases as well as assisting them in locating the nearest hospital which is well equipped to treat them. Further, it also advises the users, the precautions they should take in such scenarios. In emergency situations, the app allows the user to directly call emergency services and relates them to them the current location as well the patient details, such as name and registered medical history. It also helps users to book an appointment with desired hospitals.


Key Features:

• Simple UI/UX
• Registration
• Disgnosis
• Disease Wiki
• Locating Hospitals
• Appointment Booker

Scanit

Scanit is an android based mobile application used to extract text from an image file which is then interpreted and can be manipulated as per users need.

Description

Downloadable app apk file

Scanit is free mobile application which is used to get text from an image which then can be shared via various social media platforms such as Whatsapp, Facebook,etc. Scanit also provides an other special feature of browsing and accessing maps directly from the app itself on the extracted text.
Also, the minimalist design of Scanit makes it more easy for everyone to use. The is useful for eveyone including students, journalist, tourists, businessmen, etc


Key Features:

• Easy text extraction from image
• Simple UI/UX
• Direct Browsing facility
• Access to map directly from the app
• Sharing text via whatsapp, facebook, etc
• Saving text to local database for future use
• Copying text to clipboard facility
